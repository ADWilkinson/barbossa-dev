Barbossa

Your Autonomous AI Development Team

Barbossa is a team of AI agents that autonomously develops your software. Set it up once, and it continuously:

Discovers Work

Analyzes your codebase to find bugs, improvements, and missing features.

Creates PRs

Implements changes and submits high-quality pull requests automatically.

Reviews Code

Tech Lead agent reviews PRs, merges good ones, closes bad ones.

Runs 24/7

Works continuously on your repositories while you focus on strategy.

Quick Start

Get started in 5 minutes:

# 1. Clone the repository
git clone https://github.com/ADWilkinson/barbossa-engineer.git
cd barbossa-engineer

# 2. Copy and configure
cp config/repositories.json.example config/repositories.json
# Edit config/repositories.json with your GitHub username and repos

# 3. Run with Docker
docker-compose up -d

The Pipeline

Barbossa runs a coordinated team of 5 AI agents:

Agent Role Schedule
Product Manager Creates feature proposals 2x daily
Discovery Finds bugs and tech debt 3x daily
Engineer Implements issues, creates PRs Every 2 hours
Tech Lead Reviews and merges PRs Every 2 hours
Auditor System health checks Daily
